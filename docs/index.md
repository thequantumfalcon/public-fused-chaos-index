# Public Fused Chaos Index

This is a public-clean, offline-first release of the **Fused Chaos Index (FCI)** toolkit.

Design defaults:
- Offline-first: no implicit network.
- Public-safe: no heavyweight artifacts committed.
- Reproducible runs: timestamped run dirs + JSON manifests + SHA256 provenance.

Start with the CLI:

```bash
fci --help
fci suite run --profile offline
```
